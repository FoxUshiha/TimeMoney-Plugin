# TimeMoney-Plugin
A minecraft plugin that turns your time in money to spend and use as coins with players.

Dependencies: 

Vault: https://www.spigotmc.org/resources/vault.34315/
PlaceholderAPI: https://www.spigotmc.org/resources/placeholderapi.6245/

Commands:

/timemoney
/timepay nick value (10:00|10m|10h|10s)

Placeholder: ⏱ %player_moneytime% ⏱

Permission: time.bypass
